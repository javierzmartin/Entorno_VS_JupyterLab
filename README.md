# Entorno_VS_Jupiterlab

Hola, estoy familiarizandome con Visual Studio Code y JupyterLab

El propósito principal de este repositorio es llevar un registro del aprendizaje dentro del bootcamp de ciencia de datos de TripleTen.

El formato principal hasta el momento son cuadernos de JupyterLab ``.ipynb``

En el archivo .gitignore se incluyen las extensiones, carpetas de mi entorno local que no son relevantes para el repo,
es interesante saber como trata git lo que se añade o no a un repositorio.
    
La estructura a fecha 14/02/2026 es la siguiente:

ENTORNO_VS_JUPYTERLAB

Subcarpetas:

- Cuadernos_Webinars - # Para guardar el contenido que se trabaje en los Webinar periodicos con el Tutor.

- Ejercicios - # Ejercicios básicos, ejemplos, pruebas, etc.

- Proyectos - # Un registro de los proyectos que voy completando del curso.

Sobre datos y confidencialidad. 
    
Los datos reflejados en el repositorio han de ser entendidos como material didáctico.

Si algún material con el que trabajo contuviera datos personales de cualquier tipo no será añadido a este repositorio, siendo tratados estos en mi entorno privado con fines exclusivamente didácticos.
